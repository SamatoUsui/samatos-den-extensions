# Samato's Den Extensions

Extensiones no oficiales para **Aniyomi** y **Tachiyomi/Mihon** basadas en [Samato's Den](https://samatoden.blogspot.com/).

## Acerca de

**Samato's Den** es un espacio dedicado al arte furry y yiff, creado para quienes disfrutan de la creatividad, la fantasia y la libertad que este genero artistico ofrece. Este proyecto busca llevar ese contenido a aplicaciones compatibles con fuentes de anime y manga, manteniendo una experiencia de navegacion simple y directa.

## Que encontraras aqui

- Una extension de **anime** para publicaciones de video.
- Una extension de **manga** para publicaciones de comics.
- Soporte para varios formatos personalizados usados en el blog.
- Un proyecto pensado para facilitar el acceso al contenido desde Aniyomi y Tachiyomi/Mihon.

## Contenido soportado

Samato's Den publica contenido variado orientado a la comunidad furry, incluyendo comics, ilustraciones y videos relacionados con tematicas como yiff, rule 34, hentai, yaoi y otras expresiones para publico adulto.

Dentro de este repositorio:

- `extension/` contiene la extension de anime para videos.
- `manga-extension/` contiene la extension de manga para comics.

## Mision

Samato's Den no busca ser solo un repositorio de arte furry, sino tambien un espacio de expresion libre. Esta integracion intenta respetar ese enfoque, ofreciendo una forma comoda de explorar el contenido del sitio desde aplicaciones lectoras compatibles.

## Aviso legal

Este proyecto esta destinado exclusivamente a personas **mayores de 18 anos**. El sitio fuente puede contener material explicito y adulto relacionado con furry/yiff. Todo el contenido pertenece a sus respectivos creadores y se comparte aqui unicamente como integracion tecnica para fines de acceso, entretenimiento y aprecio artistico.

## Estado del proyecto

- Extension de manga compilable para Tachiyomi/Mihon.
- Icono personalizado de Samato's Den incluido.
